Team Members and Slider is a plugin to create your own team member section.

== Description ==

Luzuk Team is a plugin to create your own team member section. Its an amazing tool to showcsae your team members in diffrent layouts. You can arrange them in grids or you can create beuatiful sliders. You can also customize the section as per your requirement. You can create the section easily and it will be beautiful as well as reponsive. You can add there designation with links to there social media pages. This will allow you to create a impressive showcase that will lead to a lasting impression about your team. You don't require to have a knowledge of coding to customize the section. You can add various feilds like :  Title/Name (Profile Title), Designation, Profile Details Type, Description, Profile URL, Email, Personal Web URL, Profile Details, Social links

== Installation ==

= Minimum Requirements =

* PHP version 5.4 or greater
* MySQL version 5.0 or greater

= We recommend your host supports: =

* PHP version 7.0 or greater
* MySQL version 5.6 or greater
* WordPress Memory limit of 64 MB or greater (128 MB or higher is preferred)

= Installation =

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
